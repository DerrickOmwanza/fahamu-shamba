# Weather Dashboard Background Images

This folder contains background images for the Google Weather-style farmer dashboard.

## Required Images

Add these 6 images (WebP format recommended, 100-300 KB each):

1. **sunny.webp** - Clear blue sky, sunny weather
2. **cloudy.webp** - Overcast cloudy sky
3. **rainy.webp** - Rainy weather with raindrops
4. **stormy.webp** - Dark stormy sky with lightning
5. **foggy.webp** - Misty foggy morning
6. **snowy.webp** - Snowy field (optional)

## Image Specifications

- **Format**: WebP (preferred) or PNG/JPG
- **Size**: 1920×1080px or larger
- **File size**: 100-300 KB each (compressed)
- **Style**: Farming/rural scenes preferred

## Sources

- **Unsplash.com** (free, no attribution needed)
- **Pexels.com** (free)

## Optimization

Use these tools to compress images:
- **squoosh.app** (free online)
- **tinypng.com** (free online)

## How It Works

When weather changes, the dashboard automatically applies the matching background:
- Clear/Sunny → sunny.webp
- Clouds/Overcast → cloudy.webp
- Rain/Drizzle → rainy.webp
- Thunderstorm → stormy.webp
- Mist/Fog → foggy.webp
- Snow → snowy.webp

## Testing

After adding images, refresh the dashboard and switch between sub-counties to see backgrounds change.
