                        HOW IT WORKS
================================================================================

USER ACTION:
  1. Open any phone (old or new)
  2. Dial: *123#
  3. Press Call/Send button
  4. Follow system prompts
  5. Get crop recommendation

SYSTEM RESPONSE:
  - Instant menu options appear
  - User selects by pressing numbers
  - Each selection leads to next step
  - Final recommendation displayed
  - Data saved for analytics

Method: HTTP POST
Format: application/x-www-form-urlencoded

Parameters sent by USSD gateway:
  - sessionId=abc123def456
  - phoneNumber=254712345678
  - text=1*1*2
  - serviceCode=123

Response format:
  CON [menu text]        (Continue session)
  END [final text]       (End session)
