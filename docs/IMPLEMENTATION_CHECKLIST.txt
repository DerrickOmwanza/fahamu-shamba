DO THIS IN ORDER:
─────────────────────────────────────────────────────────────────────────────

[ ] STEP 1: Install Dependencies
    Command: npm install bcryptjs jsonwebtoken
    Expected: No errors, 2 packages added
    Time: ~30 seconds

[ ] STEP 2: Update server.js (3 additions)
    
    Add this import at top:
    import { initAuthRoutes } from './auth-routes.js';
    import { initializeAuthTables } from './init-auth-tables.js';
    
    Add this after database connection:
    initializeAuthTables(db);
    
    Add this before listen():
    const authRoutes = initAuthRoutes(db);
    app.use('/api/auth', authRoutes);
    
    Add this route (after existing app.get routes):
    app.get('/login', (req, res) => {
        res.sendFile(path.join(__dirname, '..', 'frontend', 'login-register.html'));
    });
    
    Verification: server.js imports correctly, no syntax errors
    Time: ~5 minutes

[ ] STEP 3: Verify File Locations
    Ensure these files exist:
    ✓ backend/auth-routes.js
    ✓ backend/init-auth-tables.js
    ✓ frontend/login-register.html
    ✓ backend/public/landing-page-optimized.html
    
    Verification: All files in place
    Time: ~1 minute

[ ] STEP 4: Add Environment Variable
    Add to .env:
    JWT_SECRET=your_secret_key_change_in_production
    
    Verification: .env file has JWT_SECRET
    Time: ~1 minute

[ ] STEP 5: Start Server
    Command: node backend/server.js
    Expected: "✅ Connected to SQLite database"
              "✅ All authentication tables initialized successfully"
              "Server running on port 5000"
    Time: ~5 seconds
