🎯 HOW TO ACCESS

Register Farmer with Siaya Info:
  http://localhost:5000/farmer-profile-dashboard
  → Click "Register Farmer" tab
  → Select from 8 Siaya subcounties
  → Select "Lake Victoria" if needed

View Farmers by Siaya Subcounty:
  http://localhost:5000/farmer-profile-dashboard
  → Click "View Farmers" tab
  → Use subcounty filter

View Dashboards:
  Main Dashboard: http://localhost:5000/dashboard → Farmers tab
  Admin Dashboard: http://localhost:5000/admin (login required)
