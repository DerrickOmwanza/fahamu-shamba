QUICK START
================================================================================

FOR USERS:
  1. Open: http://localhost:5000/farmer-dashboard
  2. Click: 🌐 language selector (top-right)
  3. Select: Your preferred language
  4. Enjoy: Interface updates instantly!

FOR DEVELOPERS:
  1. Include: <script src="/js/translations.js"></script>
  2. Use: i18n.t('key') to get translations
  3. Switch: i18n.setLanguage('swahili')
  4. Extend: Add new keys to TRANSLATIONS object

FOR DEPLOYMENT:
  1. Copy: translations.js to /backend/public/js/
  2. Replace: farmer-dashboard.html
  3. Deploy: No other changes needed
  4. Test: Language selector works
