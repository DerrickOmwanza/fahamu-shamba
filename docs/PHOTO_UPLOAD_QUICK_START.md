## How to Upload - 3 Methods

### Method 1: Click "🖼️ Browse Files" Button
1. Scroll to bottom of form
2. See "📸 Passport Photo" section
3. Click blue "🖼️ Browse Files" button
4. Select image from your computer
5. Image preview updates instantly
6. Click "Register Farmer" at bottom

### Method 2: Click the Drag-Drop Area
1. Scroll to bottom of form
2. Click on the area that says "📤 Drag image here or click to browse"
3. File picker opens
4. Select image from your computer
5. Image preview updates
6. Click "Register Farmer" at bottom

### Method 3: Drag and Drop
1. Scroll to bottom of form
2. Open file explorer on your computer
3. Find your passport photo image
4. Drag it onto the blue dashed box (📤 area)
5. Box will highlight when you hover over it
6. Release mouse to drop
7. Image preview updates
8. Click "Register Farmer" at bottom

## What Counts as a Valid Photo

### ✓ Accepted Formats
- **JPG/JPEG** - Most common, best for photos
- **PNG** - Lossless quality, good for documents
- **GIF** - Supported but not recommended
- **WebP** - Modern format

### ✓ Size Requirements
- **Max file size:** 5 MB
- **Recommended:** 200 × 250 pixels (passport size)
- **Aspect ratio:** Portrait (taller than wide)

### ❌ Not Accepted
- Word documents (.docx)
- PDF files (.pdf)
- Text files (.txt)
- Video files (.mp4, .mov)
- Audio files (.mp3, .wav)
- Compressed files (.zip, .rar)

## Troubleshooting Photo Issues

### Issue: Image Won't Preview

**Symptom:** Selected image but nothing shows

**Solution:**
1. Try a different image file
2. Ensure image is JPG, PNG, or GIF
3. Check file size is under 5MB
4. Clear browser cache (Ctrl+Shift+Delete)
5. Reload page (F5)

### Issue: Can't Select File

**Symptom:** Button or drag area not responding

**Solution:**
1. Try different upload method
2. Refresh page (Ctrl+Shift+R)
3. Check browser is up to date
4. Try different browser

### Issue: File Size Error

**Symptom:** "Photo file size must be less than 5MB"

**Solution:**
1. Use image editing software to compress
2. Reduce image resolution
3. Use JPG instead of PNG (usually smaller)
4. Online tools: tinypng.com, compressimage.io

### Issue: File Type Error

**Symptom:** "Only image files are allowed"

**Solution:**
1. Ensure file is actually an image (not document)
2. Check file extension (.jpg, .png, .gif)
3. Try different image format
4. Rename file if extension is wrong

### Issue: Photo Preview Shows But Form Won't Submit

**Symptom:** Photo displays but submission fails

**Solution:**
1. Check all 5 required fields filled
2. Check browser console for errors (F12)
3. Try submitting without photo
4. Ensure backend is running

## Tips for Best Results

### Photo Quality Tips
- Use a clear, well-lit passport photo
- Face should be centered and frontal
- Neutral background preferred
- Professional appearance
- Recent photo (within 6 months)

### File Size Tips
- Most smartphones take 2-4 MB photos
- Save as JPG for smaller size
- Use online compressor if needed
- 500 KB - 2 MB is ideal range

### Format Tips
- JPG: Best for photographs
- PNG: Better if you need transparency
- GIF: Avoid for photos (use for simple images)
- WebP: Good but less compatible

## Getting Help

### Check Console for Errors
1. Press F12 on keyboard
2. Click "Console" tab
3. Look for red error messages
4. Take screenshot for support

### Common Error Messages

**"Only image files are allowed"**
- Solution: Select a JPG, PNG, or GIF file

**"Photo file size must be less than 5MB"**
- Solution: Compress image or select smaller file

**"Error reading image file"**
- Solution: Try different image file, clear cache
