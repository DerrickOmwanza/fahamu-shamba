================================================================================
                    TRANSLATION SYSTEM
================================================================================

Location: backend/ussd-translations.json

Structure:
{
  "LANGUAGE_SELECT": {
    "en": "English text...",
    "sw": "Swahili text...",
    "luo": "Dholuo text..."
  },
  "MAIN_MENU": {
    "en": "...",
    "sw": "...",
    "luo": "..."
  },
  ... (40+ keys total)
}

How It Works:
1. Each menu step has a translation key
2. Language code (en/sw/luo) selects the text
3. Farmer sees everything in chosen language
4. Easy to add new languages (just add language code)

Language Codes:
  'en'   → English
  'sw'   → Kiswahili
  'luo'  → Dholuo

User Selection to Code:
  1 → 'en' (English)
  2 → 'sw' (Kiswahili)
  3 → 'luo' (Dholuo)
