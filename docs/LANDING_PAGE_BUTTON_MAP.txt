═══════════════════════════════════════════════════════════════════════════════
                         LOGIN PAGE (/login)
═══════════════════════════════════════════════════════════════════════════════

The login page is SMART and auto-detects the URL parameter:

URL: /login (default)          → Shows LOGIN FORM
URL: /login?mode=login         → Shows LOGIN FORM
URL: /login?mode=signup        → Shows REGISTRATION FORM (Step 1)

REGISTRATION FLOW:
Step 1: Phone + Password       → User enters security info
Step 2: Name + Location        → User enters profile info
Complete: Account created      → Token issued → Redirect to /farmer-dashboard

LOGIN FLOW:
Phone + Password               → User authenticates
Success: Token issued          → Redirect to /farmer-dashboard
