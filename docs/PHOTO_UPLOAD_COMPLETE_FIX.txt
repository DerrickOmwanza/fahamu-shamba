HOW IT WORKS NOW:
================================================================================

USER SELECTS PHOTO:
  File picker → onchange event → previewPhoto()
  ↓
  Validate: is image? < 5MB?
  ↓
  Read as base64
  ↓
  Display preview
  ↓
  ✓ Ready to submit

USER DRAGS AND DROPS:
  Drag → dragenter/dragover → highlight
  ↓
  Drop → dispatch change event
  ↓
  onchange triggers → previewPhoto()
  ↓
  Same as file picker flow
  ↓
  ✓ Preview displays

USER SUBMITS FORM:
  Collect data + optional photo
  ↓
  If photo: validate and convert to base64
  ↓
  Send JSON to server
  ↓
  If photo error: fallback to submit without
  ↓
  ✓ Registration succeeds
