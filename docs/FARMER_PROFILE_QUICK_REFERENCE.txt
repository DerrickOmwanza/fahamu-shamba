🔌 KEY API ENDPOINTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Register Farmer:
  POST /api/farmer-profile/register
  Required: phoneNumber, firstName, lastName, subCounty, farmSize

Get Farmer:
  GET /api/farmer-profile/:farmerId
  GET /api/farmer-profile/phone/:phoneNumber

List Farmers:
  GET /api/farmer-profile?limit=20&offset=0

Search Farmers:
  GET /api/farmer-profile/search/John%20Doe

Update Farmer:
  PUT /api/farmer-profile/:farmerId

Verify Profile:
  POST /api/farmer-profile/:farmerId/verify

Statistics:
  GET /api/farmer-profile/statistics

🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Dashboard Won't Load:
  1. Check server: curl http://localhost:5000/api/test
  2. Check port:   Verify 5000 is not in use
  3. Check files:  ls backend/public/farmer-profile-dashboard.html
  4. Restart:      Kill and restart node server.js

Registration Fails:
  1. Check required fields are filled (5 required fields)
  2. Verify phone number is unique (not already registered)
  3. Check server logs for error messages
  4. Check browser console (F12) for errors

No Search Results:
  1. Verify farmers exist: GET /api/farmer-profile?limit=1
  2. Check search term matches data
  3. Try searching by phone or farmer ID
  4. Check database: sqlite3 backend/fahamu_shamba.db "SELECT COUNT(*) FROM farmer_profiles;"

API Returns 404:
  1. Verify URL is correct: /api/farmer-profile/...
  2. Check server is running: curl http://localhost:5000/api/test
  3. Verify routes are registered in server.js
  4. Restart server: Kill and restart node
