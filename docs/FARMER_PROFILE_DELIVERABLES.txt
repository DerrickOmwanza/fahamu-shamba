3. API ENDPOINTS (12 Total)
================================================================================

✅ POST   /api/farmer-profile/register
   - Register new farmer with complete profile
   - Auto-generates unique farmer ID
   - Calculates profile completion percentage
   - Returns success/error response

✅ GET    /api/farmer-profile/:farmerId
   - Retrieve complete farmer profile
   - Includes associated farms and activity logs
   - Returns 404 if not found

✅ GET    /api/farmer-profile/phone/:phoneNumber
   - Retrieve farmer by phone number
   - Fast lookup using indexed field

✅ PUT    /api/farmer-profile/:farmerId
   - Update farmer profile information
   - Partial updates supported (any field optional)
   - Recalculates profile completion
   - Logs activity

✅ GET    /api/farmer-profile
   - List all farmers with pagination
   - Supports filtering (sub-county, soil type, verified status)
   - Returns pagination metadata
   - Limit and offset parameters

✅ GET    /api/farmer-profile/search/:searchTerm
   - Full-text search across multiple fields
   - Searches name, phone, email, farmer ID
   - Returns up to 50 matching results

✅ POST   /api/farmer-profile/:farmerId/verify
   - Mark farmer profile as verified
   - Logs verification with timestamp
   - Updates verification status

✅ POST   /api/farmer-profile/:farmerId/deactivate
   - Deactivate farmer profile
   - Accepts optional deactivation reason
   - Logs deactivation event

✅ POST   /api/farmer-profile/:farmerId/reactivate
   - Reactivate deactivated profile
   - Logs reactivation event

✅ GET    /api/farmer-profile/statistics
   - Returns comprehensive statistics
   - Total farmers, verification rate
   - Distribution by location and agriculture
   - Farm size and budget statistics

✅ GET    /api/farmer-profile/export
   - Export farmer data (JSON format)
   - Supports filtering by sub-county and soil type
   - Returns complete farmer records
