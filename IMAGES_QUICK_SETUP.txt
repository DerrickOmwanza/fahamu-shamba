========================================
WEATHER DASHBOARD - QUICK IMAGE SETUP
========================================

You have 2 options to add weather background images:

OPTION 1: EASY (5 minutes)
==========================

1. Go to Unsplash.com or Pexels.com

2. Search for and download these 6 images:
   - "sunny weather blue sky" → sunny.webp
   - "overcast cloudy field" → cloudy.webp
   - "rainy weather rain drops" → rainy.webp
   - "thunderstorm dark sky" → stormy.webp
   - "foggy mist morning" → foggy.webp
   - "snowy field snow" → snowy.webp (optional)

3. Compress images:
   - Use squoosh.app to convert to WebP (80-150 KB each)
   - Or use tinypng.com to compress PNG/JPG

4. Create folder in your project:
   c:/Users/ADMIN/fahamu-shamba1-main/public/images/

5. Copy all 6 images into that folder

6. Git commit:
   git add public/images/
   git commit -m "Add weather dashboard background images"
   git push

DONE! Dashboard will now show dynamic weather backgrounds.

IMAGE SPECS
===========

- Format: WebP (preferred) or PNG/JPG
- Size: 1920×1080px or larger
- File size: 100-300 KB each
- Compressed: Yes (use squoosh.app or tinypng.com)


HOW IT WORKS
============

When weather changes:
1. Backend returns weather condition (e.g., "Rain")
2. JavaScript applies CSS class (e.g., .rainy-bg)
3. Background image from /images/rainy.webp displays
4. Dark overlay (30% opacity) ensures text readability
5. Hero card, hourly strip, and detail cards overlay the image
6. Smooth fade transition (0.5s) when switching
